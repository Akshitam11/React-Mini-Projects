# Mini Projects Hub

This project combines multiple independent React mini projects into one Vite app using `react-router-dom`.

The home page lists all mini projects, and each one opens on its own route through the shared navbar. The original project logic was kept as separate components and mounted inside the main app.

Included mini projects:

- Drum Kit
- Mouse Position
- Window Resize
- Toggle Theme
- Custom Hook
- Online/Offline Status
- Form with `useRef`

Run locally with:

```bash
npm install
npm run dev
```
